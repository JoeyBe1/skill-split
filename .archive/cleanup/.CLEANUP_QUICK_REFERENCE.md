## Execution (4 Simple Steps)

### Step 1: Create Archive Directories
```bash
mkdir -p .archive/{wave-reports,phase-plans,task-reports,testing-docs,test-reports,investigations,strategy-docs,user-guides}
```

### Step 2: Move Files to Archive

**Wave reports**:
```bash
mv WAVE_*.md .archive/wave-reports/
```

**Phase plans**:
```bash
mv PHASE_*.md .archive/phase-plans/
```

**Task reports**:
```bash
mv HAIKU_EXECUTION_TASKLIST.md FINAL_GAP_CLOSURE_TASKS.md TASK-011-*.md TEST_EXECUTION_SUMMARY.md .archive/task-reports/
```

**Roundtrip tests**:
```bash
mv ROUNDTRIP_*.md START_HERE_ROUNDTRIP.md TEST_ROUNDTRIP_README.md .archive/testing-docs/
```

**Test reports**:
```bash
mv COMPREHENSIVE_TEST_REPORT.md REAL_COMPONENT_TEST_REPORT.md REAL_FILE_TEST_REPORT.md SKILL_COMPOSITION_TEST_REPORT.md .archive/test-reports/
```

**Investigations**:
```bash
mv DB_COUNT_INVESTIGATION.md FOLDER_RECREATION_INVESTIGATION.md SYNC_*.md ARCHITECTURE_COMPARISON.md COMPOSITION_*.md HANDLER_FIXES_2026_02_04.md .archive/investigations/
```

**Strategy docs**:
```bash
mv IMPLEMENTATION_COMPLETE.md PRODUCTION_TEST_PLAN.md SESSION_SUMMARY.md HANDOFF.md .archive/strategy-docs/
```

**Guides**:
```bash
mv JOEY_GUIDE.md LIBRARIAN_WORKFLOW.md RALPH_USAGE.md IDEAS.md ARCHITECTURE.md .archive/user-guides/
```

### Step 3: Create Archive Index

Create `.archive/README.md`:
```markdown

- **phase-plans/** - Phase planning documents
- **task-reports/** - Task execution records

- **strategy-docs/** - Strategy documentation
- **user-guides/** - Developer workflow guides

See root documentation for current docs.
```

