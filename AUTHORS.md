## Project Origins

skill-split was created to solve the progressive disclosure problem for AI documentation workflows. It enables:

- 99% token savings through section-level loading
- Byte-perfect round-trip integrity
- Multi-mode search (BM25, Vector, Hybrid)
- Production-ready SQLite and Supabase integration
